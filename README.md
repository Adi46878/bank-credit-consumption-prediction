# Bank Credit Consumption Prediction

## 📌 Business Problem

A leading bank wants to predict the **average credit card consumption** of customers for the next 3 months based on their past behavior and demographics.

---

## 📊 Dataset

The dataset includes:

* Customer Demographics
* Customer Behavioral Data
* Credit Card Consumption (Target Variable)

---

## ⚙️ Approach

* Data Merging and Cleaning
* Handling Missing Values
* Encoding Categorical Variables
* Train-Test Split
* Model Building using Random Forest Regressor

---

## 📈 Model Evaluation

Evaluation Metric used: **RMSPE (Root Mean Square Percentage Error)**

**RMSPE Score:** `0.2877`

---

## 🤖 Model Used

* Random Forest Regressor

---

## 📁 Files in Repository

* `customer_credit_consumption_model.ipynb` → Complete code
* `Final_Predictions.csv` → Predicted values for missing customers

---

## 🚀 Conclusion

The model is able to predict customer credit consumption with reasonable accuracy and can help the bank in targeted marketing and customer segmentation.
